# 🏏 IPL Data Analysis using Python

An Exploratory Data Analysis (EDA) project on the Indian Premier League (IPL) dataset to uncover insights into team performance, player statistics, batting trends, bowling performance, venue impact, and season-wise trends using Python.

---

## 📖 Project Overview

This project analyzes historical IPL data from **matches.csv** and **deliveries.csv** datasets. The objective is to transform raw cricket data into meaningful insights through data cleaning, feature engineering, visualization, and statistical analysis.

---

## 🎯 Objectives

- Analyze overall IPL match statistics
- Evaluate team performance across seasons
- Identify top batsmen and bowlers
- Study venue-wise performance
- Analyze toss impact on match results
- Discover season-wise scoring trends
- Generate actionable insights using visualizations

---

## 📂 Dataset

The project uses two datasets:

- **matches.csv** – Match-level information
- **deliveries.csv** – Ball-by-ball delivery data

## 📌 Key Insights

- Winning the toss provides only a slight advantage.
- Batting has become increasingly aggressive over the years.
- Home venues often improve team performance.
- A small number of players dominate batting and bowling statistics.
- Run rates have steadily increased across IPL seasons.
